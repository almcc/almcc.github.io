e:T828,<p>Google App Engine holds onto older versions of your app after you push a new deployment, however there are limits (20) to how many you can keep. If you hit this limit any new pushes will fail to deploy. I wanted my Continuous Deployment pipeline to keep the last 3 versions of a service. The following bash script will do this for me:</p>
<pre><code class="hljs language-bash">*#!/bin/bash*

VERSIONS**=**$(gcloud app versions list --service <span class="hljs-variable">$1</span> --sort-by <span class="hljs-string">'~version'</span> --format <span class="hljs-string">'value(version.id)'</span>)
COUNT**=**0
<span class="hljs-built_in">echo</span> <span class="hljs-string">"Keeping the <span class="hljs-variable">$2</span> latest versions of the <span class="hljs-variable">$1</span> service"</span>
**<span class="hljs-keyword">for</span>** VERSION **<span class="hljs-keyword">in</span>** <span class="hljs-variable">$VERSIONS</span>
**<span class="hljs-keyword">do</span>((**COUNT++**))<span class="hljs-keyword">if</span>** **[** <span class="hljs-variable">$COUNT</span> -gt <span class="hljs-variable">$2</span> **]<span class="hljs-keyword">then</span>**
      <span class="hljs-built_in">echo</span> <span class="hljs-string">"Going to delete version <span class="hljs-variable">$VERSION</span> of the <span class="hljs-variable">$1</span> service."</span>
      gcloud app versions delete <span class="hljs-variable">$VERSION</span> --service <span class="hljs-variable">$1</span> -q
    **<span class="hljs-keyword">else</span>**
      <span class="hljs-built_in">echo</span> <span class="hljs-string">"Going to keep version <span class="hljs-variable">$VERSION</span> of the <span class="hljs-variable">$1</span> service."</span>
    **<span class="hljs-keyword">fi</span>
<span class="hljs-keyword">done</span>**
</code></pre>
<p>You can call it like this:</p>
<pre><code class="hljs language-bash">bash delete-older-gcloud-app-versions.sh default 3
</code></pre>
